## Contexto del Proyecto

### Tema
**Sistema de Automatización mediante RPA para Optimización de Procesos Productivos SAP en Entorno Industrial en Maflow Spain automotive.**

### Descripción
El objetivo principal de este proyecto es analizar, diseñar e implementar soluciones de **Automatización Robótica de Procesos (RPA)** para interactuar con el sistema ERP **SAP** utilizado en **Maflow Spain Automotive**.

La implementación busca:
1.  Reducir la carga manual en tareas repetitivas de gestión de producción.
2.  Minimizar errores humanos en la introducción de datos en SAP y en el entorno del MES.
3.  Optimizar los tiempos de ciclo de los procesos administrativos vinculados a planta.

### Metodología
Este proyecto sigue la metodología **RUP (Rational Unified Process)**, estructurando el desarrollo en cuatro fases iterativas:
*   **Inicio (Inception)**: Definición del alcance y objetivos.
*   **Elaboración (Elaboration)**: Análisis de riesgos y arquitectura.
*   **Construcción (Construction)**: Desarrollo de los bots.
*   **Transición (Transition)**: Despliegue y validación en entorno real.
