# Sistema de Automatización RPA para Optimización de Procesos SAP en Maflow Spain Automotive

![Estado](https://img.shields.io/badge/Estado-Completado-success)
![Metodología](https://img.shields.io/badge/Metodología-RUP-blue)
![Stack](https://img.shields.io/badge/Stack-React%20%7C%20Node%20%7C%20UiPath%20%7C%20SQLServer-orange)
![Universidad](https://img.shields.io/badge/Universidad-UNEATLANTICO-red)

Este repositorio contiene la documentación técnica completa del Trabajo de Fin de Grado (TFG) enfocado en la automatización de procesos industriales mediante RPA (Robotic Process Automation) para la generación de documentos Galia en el entorno SAP de Maflow Spain Automotive.

## 📁 Estructura de Fases RUP

El proyecto sigue la metodología RUP (Rational Unified Process), dividida en cuatro fases fundamentales:

| Fase | Contenido | Estado |
|------|-----------|--------|
| [📋 01 · Inicio](./01-inicio/README.md) | Visión, modelo dominio, actores, riesgos, glosario | ✅ |
| [🔬 02 · Elaboración](./02-elaboracion/README.md) | Arquitectura, diagramas estado, casos de uso, requisitos | ✅ |
| [🔨 03 · Construcción](./03-construccion/README.md) | Diseño MVC, paquetes, BBDD, UiPath, solución | ✅ |
| [🚀 04 · Transición](./04-transicion/README.md) | Pruebas, resultados, métricas, conclusiones | ✅ |

## 🚀 Cómo usar este repositorio

Este repositorio está diseñado para ser navegado de forma interactiva:
1. Utiliza la **barra de navegación superior** en cualquier página para saltar entre fases.
2. Despliega el menú **"Ver todas las secciones"** para acceder a detalles específicos.
3. Sigue el orden de las fases (01 a 04) para entender el ciclo de vida completo del proyecto.
