# Diagramas y Modelado

Repositorio visual de todos los diagramas generados durante el ciclo de vida del proyecto RUP.

## Tipos de Diagramas
1. **Diagramas de Casos de Uso**: Actores y funcionalidades.
2. **Diagramas de Actividad**: Flujo de trabajo del proceso industrial en Maflow.
3. **Diagramas de Secuencia**: Interacción paso a paso entre el Robot y SAP.
4. **Diagramas de Clases/Componentes**: Estructura del proyecto.

> *Nota: Las imágenes de los diagramas se almacenarán en la carpeta `/img`.*
