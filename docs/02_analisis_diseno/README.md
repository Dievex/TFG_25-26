# Análisis y Diseño 

Esta sección detalla cómo se realizará técnicamente la solución RPA para SAP.

## Artefactos Principales
- **Documento de Arquitectura de Software**: Estructura del robot y su integración con SAP.
- **Diseño de Procesos**: Flujogramas del proceso actual (AS-IS) y el proceso automatizado (TO-BE).
- **Modelo de Datos**: Si aplica, estructuras de datos utilizadas por el bot.

## Herramientas
- Herramienta RPA: UiPath
- SAP ERP
