# Requisitos

En esta sección se documentan los requisitos del sistema siguiendo la metodología RUP.

## Artefactos Principales
- **Visión del Proyecto**: Alcance y objetivos generales.
- **Especificación de Requisitos de Software (SRS)**: Requisitos funcionales y no funcionales.
- **Casos de Uso**: Descripción detallada de las interacciones usuario-sistema.
- **Glosario**: Terminología específica de Maflow, SAP y RPA.
