# Gestión del Proyecto y Metodología

Seguimiento de las fases e hitos del proyecto.

## Fases RUP
1. **Inception (Inicio)**: Definición del alcance y viabilidad.
2. **Elaboration (Elaboración)**: Análisis de riesgos y arquitectura base.
3. **Construction (Construcción)**: Desarrollo iterativo de los bots RPA.
4. **Transition (Transición)**: Despliegue en entorno productivo de Maflow y formación.
